## 荆南俊俊体

- **font-family**: `JUNJUN`
- **风格**: 粗黑手写体，像粗头马克笔写出来的效果，笔画粗壮有力且不规则
- **来源**: chinese-fonts-cdn
- **原址**: https://chinese-font.netlify.app/zh-cn/

### CDN

```html
<link rel="stylesheet" href="https://chinese-fonts-cdn.deno.dev/packages/jnjj/dist/JUNJUN/result.css">
```

```css
h1 { font-family: 'JUNJUN', sans-serif; }
```

## 字魂扁桃体

- **font-family**: `zihunbiantaoti`

- **来源**: chinese-fonts-cdn / ZeoSeven
- **原址**: https://fonts.zeoseven.com/items/405/

### CDN

```html
<link rel="stylesheet" href="https://chinese-fonts-cdn.deno.dev/packages/zhbtt/dist/字魂扁桃体/result.css">
```
