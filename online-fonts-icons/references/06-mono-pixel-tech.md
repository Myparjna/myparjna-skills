# ⑥ Mono / Pixel / Tech / 等宽像素科技风格（5 款）

适合代码编辑器、终端界面、游戏 UI、复古终端风格等场景。

---

## Maple Mono CN Thin

- **font-family**: `Maple Mono CN Thin`
- **风格**: 超细等宽代码字体，中文支持好，笔画纤细精致
- **来源**: chinese-fonts-cdn
- **原址**: https://font.subf.dev/zh-cn/ / https://github.com/subframe7536/maple-font

### CDN

```html
<link rel="stylesheet" href="https://chinese-fonts-cdn.deno.dev/packages/maple-mono-cn/dist/MapleMono-CN-Thin/result.css">
```

```css
code { font-family: 'Maple Mono CN Thin', monospace; }
```
