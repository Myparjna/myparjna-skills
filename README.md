# Myparjna Skills

Reusable skills for Codex, Claude Code, OpenCode, and compatible agent runtimes.

## Included skills (10)

| Skill | 用途 |
|-------|------|
| `project-handoff` | 项目交接文档生成（新版） |
| `see-tools` | 工具查看与管理 |
| `modelscope-search` | 魔搭社区模型搜索 |
| `online-fonts-icons` | 在线中文字体与图标库 CDN 集成 |
| `skillsmp-search` | 技能市场搜索 |
| `amap-cli-skill` | 高德地图 CLI 工具 |
| `amap-jsapi-skill` | 高德地图 JS API 集成指南 |
| `amap-lbs-skill` | 高德地图 LBS 服务 |
| `harmony-ux-check` | 鸿蒙 UX 规范检查 |
| `华为体验手册skill_extracted` | 华为体验手册资料库 |

## Install

Install one skill:

```bash
npx skills add https://github.com/Myparjna/myparjna-skills --skill project-handoff --yes
npx skills add https://github.com/Myparjna/myparjna-skills --skill online-fonts-icons --yes
```

List available skills:

```bash
npx skills add https://github.com/Myparjna/myparjna-skills --list
```
